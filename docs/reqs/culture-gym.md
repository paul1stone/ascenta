# Culture Gym

> Standalone Feature | Daily Behavioral Learning System | Employee | Manager | HR / People Ops

## Overview

Culture Gym is the daily behavioral learning system inside Ascenta that turns workplace culture into consistent habits through short daily skill practice. Each day, employees complete a 5-7 minute Culture Gym Workout that teaches, applies, and measures workplace skills. Culture Gym trains 365 measurable workplace skills per year.

**Tagline:** Teach it. Apply it. Measure it. Improve it.

Culture Gym is a standalone feature within Ascenta, separate from Grow.
