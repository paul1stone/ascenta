Create a Custom Goal with AI Assistance
Open the Do tab to describe a goal in your own words. The AI will use the same MVV, JD, and strategy context to help structure and refine it.

Three Source Inputs

The Learn tab aggregates company MVV, the employee's job description, and Strategy Studio OKRs into a unified context. This ensures goal recommendations are grounded in both organizational direction and role-specific expectations.

AI-Powered Recommendations

Each recommendation shows which sources informed it (color-coded tags), a confidence score, and a brief rationale. Managers can select multiple recommendations to batch-send to the Do tab for full goal creation.

Learn → Do Flow

Selected recommendations pre-fill the goal creation form in the Do tab. The "Create a Custom Goal" option opens the Do chat with the same context injected, so open-ended goals still benefit from MVV, JD, and strategy awareness.

Strategy Studio Bridge

The Plan → Grow connection is explicit: Strategy Studio OKRs from the Plan section are surfaced as a source alongside MVV and JD. This closes the loop between organizational strategy and individual goal-setting.
