Performance Review — Chat Flow & AI Tools

How Compass guides the manager through each step via working document

AI Tools (4 new tools in grow-tools.ts)

1. startPerformanceReview
Input: employeeName, employeeId, reviewPeriod
Action: Creates PerformanceReview record, pulls goals/check-ins/notes/foundation for the period, pre-fills contribution suggestions
Opens: Working doc at Step 2 (contributions form) with context sidebar
2. generateReviewDraft
Input: reviewId (contributions already saved)
Action: AI synthesizes pulled data + manager contributions into review sections, frames language around strategic pillars/values
Opens: Working doc at Step 3 (draft preview, read-only)
3. finalizeReview
Input: reviewId + edited sections
Action: Copies draft → finalDocument with manager edits, sets status to "finalized", records signoff
Opens: Working doc at Step 4 (editable final version)
4. recommendNextGoals
Input: reviewId
Action: AI analyzes review + current strategy to suggest 2-4 next-period goals with rationale
Opens: Working doc at Step 5 (goal recommendations list with "Create Goal" handoff buttons)

Chat Conversation Flow

Manager says: "Start a performance review for Sarah Chen"

1
Pull Context (automatic)
Compass calls startPerformanceReview → fetches goals, check-ins, notes, and foundation data for Sarah's review period. Opens working document with contributions form.
💬 "I've pulled Sarah's Q1 data — 4 goals (3 on track, 1 completed), 3 check-ins, and 2 performance notes. I've pre-filled what I can from the data. Please review the contributions section."
2
Manager Contributions (interactive)
Working doc shows: pre-filled strategic priorities & outcomes from goal data + empty fields for behaviors/additional context. Sidebar shows pulled goals, notes, and which pillars Sarah ladders to. Manager edits/completes the form, or asks Compass to help.
💬 Manager: "She also led the Q1 onboarding redesign" → Compass updates contributions via updateWorkingDocument
3
AI Drafts Review (generation)
Manager submits contributions → Compass calls generateReviewDraft. AI writes: summary, strengths & impact, areas for growth, strategic alignment narrative, overall assessment. Language tied to company pillars/values.
💬 "Here's the draft review. The strengths section highlights Sarah's contributions to our 'Customer Obsession' pillar through the onboarding redesign. Take a look and let me know what to adjust."
4
Manager Reviews & Finalizes (editable)
Working doc switches to editable mode — all draft sections become text areas. Manager can edit directly or ask Compass: "Make the growth section more specific about presentation skills." When satisfied, clicks Finalize.
💬 "Review finalized. You can download it as a PDF. Would you like to set next-period goals for Sarah?"
5
Goal Recommendations (optional handoff)
If manager says yes → Compass calls recommendNextGoals. Shows 2-4 suggested goals based on review + current strategy. Each has a "Create This Goal" button that hands off to existing startGoalCreation tool.
If "not right now" → Compass saves recommendations to the review record for later. Manager can revisit anytime.

Working Document Form Phases

The side-panel form adapts per step

Steps 1-2: Input
• Context sidebar (goals, notes, pillars)
• Pre-filled contributions form
• Editable text areas for subjective input
• "Generate Draft" button when ready
Steps 3-4: Review & Edit
• Read-only draft preview (Step 3)
• Switches to editable text areas (Step 4)
• Section-by-section editing
• "Finalize" + "Download PDF" buttons
Step 5: Goals
• Recommended goals list
• Rationale per recommendation
• "Create This Goal" buttons
• "Save for Later" option
